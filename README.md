该项目为落雪咖啡屋 maimai DX 查分器的前端部分，使用 React + TypeScript + Vite 编写。

了解查分器的使用方法，请访问：[官方文档](https://maimai.lxns.net/docs)

## 特性

- 📱 支持舞萌 DX 与中二节奏的游戏数据同步
- 💻 不受设备限制，支持多平台同步、管理游戏数据
- 🚀 快速的游戏数据同步体验
- 🤩 易于使用的成绩、曲目与姓名框查询功能
- 🌐 开放的 API 接口，支持第三方开发者对接
- 🌈 支持暗色与自动主题
